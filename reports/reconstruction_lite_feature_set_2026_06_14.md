# reconstruction_lite Feature Set

Run date: 2026-06-14

Generated by `scripts/build_reconstruction_lite_feature_report.py` from `forensic_compare.conventional.feature_names("reconstruction_lite")`.

`reconstruction_lite` is a standalone AEROBLADE-lite style ablation for the current single-image benchmark setup. It does not run a learned diffusion or autoencoder reconstructor; instead, it measures deterministic bicubic downsample/upsample residuals at half and quarter scale, then adds cross-scale deltas. That keeps the branch fast, deterministic, and separable from `combined_v4`.

## How To Run

```powershell
python scripts/run_feature_baseline.py `
  --data-dir data/raw/ishu-2024-ai-generated-vs-real-images/dataset `
  --output-dir runs/reconstruction_lite_probe/seed7 `
  --feature-set reconstruction_lite `
  --classifier logistic_regression `
  --seed 7
```

For a physics-guided neural ablation, pass `--physics-feature-set reconstruction_lite` to `scripts/train_physics_guided_net.py`. The existing `combined_v3` and `combined_v4` definitions are unchanged, so old metrics remain comparable.

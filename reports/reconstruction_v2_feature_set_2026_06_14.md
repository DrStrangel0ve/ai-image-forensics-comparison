# reconstruction_v2 Feature Set

Run date: 2026-06-14

Generated by `scripts/build_reconstruction_v2_feature_report.py` from `forensic_compare.conventional.feature_names("reconstruction_v2")`.

`reconstruction_v2` extends `reconstruction_lite` with deterministic compression-style residuals. It adds FFT low-pass reconstructions and per-channel low-rank SVD reconstructions so the project can test autoencoder-like residual signals without downloading a learned reconstructor. This is still an ablation branch; it is not a claim of AEROBLADE/FIRE parity.

## How To Run

```powershell
python scripts/run_feature_baseline.py `
  --data-dir data/raw/ishu-2024-ai-generated-vs-real-images/dataset `
  --output-dir runs/reconstruction_v2_probe/seed7 `
  --feature-set reconstruction_v2 `
  --classifier logistic_regression `
  --seed 7
```

For a physics-guided neural ablation, pass `--physics-feature-set reconstruction_v2` to `scripts/train_physics_guided_net.py`. Historical `reconstruction_lite`, `combined_v3`, and `combined_v4` definitions are unchanged.
